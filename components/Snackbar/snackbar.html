Snackbar

Snackbars inform users of a process that an app has performed or will perform. They appear temporarily, towards the bottom of the screen. They shouldn’t interrupt the user experience, and they don’t require user input to disappear.

Simple Snackbar

This can be used by using class snackBar , for text use class snackbar--text , for message give class exit .

Can't take photo . Retry in 5 seconds

RETRY 
        
<div class="snackBar">
    <p class="snackbar--text p0">Can't take photo . Retry in 5 seconds</p>
    <div class="exit"><span class=" bold ">RETRY</span> 
        <i class="fas fa-times text_white"></i>
    </div>

Stacked Snackbar

This can be used by using class snackBar--stacked , for text use class snackbar--text , for message give class exit--stacked .

Can't take photo . Retry in 5 seconds

RETRY 

<div class="snackBar--stacked br-0-5">
    <p class="snackbar--text p0">Can't take photo . Retry in 5 seconds</p>
    <div class="exit--stacked"><span class=" bold ">RETRY</span>
        <i class="fas fa-times text_white"></i></div>
</div>
