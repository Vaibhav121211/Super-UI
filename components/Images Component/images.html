Images

Super UI Gives You access of the responsive Images as well as Round Images .

Responsive Images

This Can be used every where , when ever you need a responsive image . You can use it by adding image--component in parent div , and class image--responsive in image tag . 

      
<div class="image--component">

<img class="image--responsive"
src="/Documentations/Components/Images/imges/willian-justen-de-vasconcellos-RzxMY9_kAC4-unsplash.jpg" alt="waves">

</div>

Round Image

This Can be used every where , when ever you need a responsive round image . You can use it by adding image--component__round in parent div , and class image--responsive in image tag . 

        
<div class="image--component__round">

<img class="image--responsive" 
src="/Documentations/Components/Images/imges/willian-justen-de-vasconcellos-RzxMY9_kAC4-unsplash.jpg" alt="waves">
</div>
