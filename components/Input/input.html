Input

Input's are used everywhere in websites , So Super UI is providing various inputs to you incorporate in your websites .

TextBox

This can be used as message box , or a contact me form in your website . You can use it by using class textbox--container in the parent and write yur message in textarea tag. 


                  
      <div class="textbox--container">
      
      <textarea cols="30" rows="10" >Write Something Here !!</textarea>
      
      <button class="btn btn--icon btn--primary" ><i class="fas fa-paper-plane"></i>Send</button>

Input with Validation and Error Style

These types of inputs can be used at Login page or sign up page . They can be used as , given below code snippets .

You have to fill this .
This email exists
Login Successful
                
      <div class="input--text">
      <input class="input input--status__none" type="text" placeholder="Enter Something">
      <div class="input--status__message--none">You have to fill this .</div>
      </div>
                  
                  
      <div class="input--text">
      <input class="input input--status__error" type="text" placeholder="Enter Something">
      <div class="input--status__message--error">This email exists</div>
      </div>
                  
      <div class="input--text">
      <input class="input input--status__success" type="text" placeholder="Enter Something">
      <div class="input--status__message--success">Login Successful</div>
      </div>
      </div>
