List

List are very Useful if you want to give some information about your services , or if u want some categories inside your websites .

Spaced (Normal List)

This can be used by makind a div having class list--container and then you have to use class list--heading then type your list items and super ui will do everything for You .

Brands

  • Rolex
  • Hublot
  • Montblanc
  • Omega

<div class="list--container">
<p class="list--heading">Brands</p>
<ul>
    <li>Rolex</li>
    <li>Hublot</li>
    <li>Montblanc</li>
    <li>Omega</li>
</ul>
</div>
