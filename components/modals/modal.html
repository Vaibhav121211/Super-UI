Modals

A modal (also called a modal window or lightbox) is a web page element that displays in front of and deactivates all other page content. To return to the main content, the user must engage with the modal by completing an action or by closing it.

Simple Modal

This will make your modal very easily , just add class modal--simple and for buttons add class modal--btn.

Simple Question

Some Conditions


    
<div class="modal--simple p1">
    <p class="h3 p2">Simple Question  </p>
    <p class="h2 txt--grey  p1">Some Conditions</p>
    <div class="modal--btn">
    <button class="btn btn--info ">Action 1</button>
    <button class="btn btn--info"> Action 2</button>
    </div>
</div>

Stacked Modal

This will make your modal very easily , just add class modal--stacked and for buttons add class modal--stacked__btn.

Simple Question

Some Conditions


    
<div class="modal--stacked p1">
    <p class="h3 p2">Simple Question  </p>

    <div class="modal--stacked__btn">
    <button class="btn btn--info ">Action 1</button>
    <button class="btn btn--info"> Action 2</button>
    </div>
</div>
