Grids

Super ui allows you to incorporate grids in your websites by using our grids. You can use it to make your website responsive . Just by adding grids classes that are shown below .

Two items in a grid .

This can be used by using grid--container--2 class on the container . And use class grid--item--2 into the grid items .
